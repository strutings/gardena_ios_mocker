# Gardena iOS Mocker for Home Assistant

This is a custom Home Assistant integration that mimics the official Gardena iOS app's behavior, allowing direct communication with the Gardena/Husqvarna BFF API. It bypasses certain endpoint limitations and provides stable control for both robotic mowers and irrigation systems.

## Features
- **Lawn Mower Control**: Start, pause, and dock commands mimicking iOS traffic.
- **Advanced Mower Settings**: Configure remote starting points, proportions, and boundary wire settings directly.
- **Smart Irrigation Control**: Direct manual valve overrides bypassing standard command layer limits.
- **Multi-language Support**: Includes translations for English, Norwegian, German, French, Dutch, Slovak, Swedish, and Finnish.
